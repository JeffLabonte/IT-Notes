# MongoDB Notes



## Show databases

<u>You need to be connected as admin</u> 

` show dbs`

output : 

```mongo output
> show dbs
admin       0.000GB
config      0.000GB
local       0.000GB
weather_ai  0.000GB
```



## Select a db

` use weather_ai`

```
> use weather_ai
switched to db weather_ai
```
